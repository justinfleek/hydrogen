# One-Shot Embroidery Customization via Contrastive LoRA Modulation

**Authors:** Jun Ma, Qian He, Gaofeng He, Huang Chen, Chen Liu, Xiaogang Jin, Huamin Wang (Zhejiang Sci-Tech University, Style3D Research, Zhejiang University)  
**Publication:** SIGGRAPH 2025  
**Date:** 2025  
**Domain:** Computer Graphics / Diffusion Models / Textile Synthesis

---

## Abstract

One-shot embroidery customization using **contrastive LoRA modulation**.

Given a reference embroidery image, generates novel embroidery from:
- Image inputs
- Textual descriptions

Uses **contrastive learning** to disentangle fine-grained style and content.
