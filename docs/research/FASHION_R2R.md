# FashionR2R: Texture-preserving Rendered-to-Real Image Translation with Diffusion Models

**Authors:** Rui Hu, Qian He, Gaofeng He, Jiedong Zhuang, Huang Chen, Huafeng Liu, Huamin Wang (Zhejiang University, Style3D Research)  
**Publication:** NeurIPS 2024  
**Date:** 2024  
**Domain:** Computer Vision / Image Translation / Fashion

---

## Abstract

Translates **rendered fashion images to realistic counterparts** using diffusion models.

**Two-stage framework:**
1. **DKI (Domain Knowledge Injection):** Positive/negative domain finetuning
2. **RIG (Realistic Image Generation):** Texture-preserving attention control (TAC)

**Key Innovation:** Preserves fine-grained clothing textures while enhancing photorealism.
